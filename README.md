# 💰 వడ్డీ లెక్కలు (Interest Calculator)

Modern Telugu Interest Calculator optimized for **Node.js 22**.

## 🚀 Two Ways to use this as an "App"

### Option 1: The "Small Thing" Way (PWA - Recommended)
Since you don't want a heavy setup, this is the best way:
1. Open your website URL in **Chrome** on your Android phone.
2. Tap the **three dots (⋮)** in the top right.
3. Tap **"Install App"** or **"Add to Home Screen"**.
4. It will now appear on your home screen with its own icon, work offline, and open without the browser bar. **No APK needed!**

### Option 2: The "Cloud Build" Way (Generate APK without Android Studio)
Use GitHub to build the APK for you:
1. Create a repository on **GitHub** and push this code.
2. Go to the **"Actions"** tab in your GitHub repository.
3. You will see a workflow named **"Build Android APK"** running.
4. Once it finishes (about 3 minutes), click on the build run, scroll down to **Artifacts**, and download `vaddi-lekhalu-apk`.
5. Unzip it, and you have your `.apk` file!
